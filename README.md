# 🎥 YouTube Storytelling Planner

A persistent, responsive web application designed for content creators to brainstorm, organize, and track the production status of faceless video concepts. 

 **[Click Here to Launch the Live Web App](https://KirkFranklin.github.io/youtube-story-planner/)
---

## Features

* **Data Persistence:** Integrated with browser `localStorage` and JSON state management to ensure video ideas are never lost on page refresh.
* **Real-Time Search Filter:** Instant client-side search functionality filtering through titles and plot notes simultaneously.
* **Multi-Criteria Sorting:** Live genre dropdown filtering built using combined logical states.
* **Interactive Production Pipeline:** Dynamic, inline status updating that instantly alters card badges and component UI styling.
* **Live Analytics Dashboard:** Real-time data aggregator tracking total video ideas and calculating active totals for each production stage.
* **Base64 Image Processing:** Utilizes JavaScript `FileReader` API to convert uploaded thumbnail concept drafts into text strings for browser-side database storage.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic markup structure.
* **CSS3:** Custom properties (CSS variables), grid layout, flexbox, and mobile-responsive media queries.
* **JavaScript (ES6+):** Event listeners, array filtering (`.filter()`), dynamic DOM manipulation, and Web Storage API.
